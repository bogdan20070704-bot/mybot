# 🏰 Подземелье и Уровни - Telegram RPG Bot

Полноценная RPG-игра в Telegram с системой уровней, подземельями, PvP боями, карточной системой предметов, гильдиями, рынком, турнирами, квестами и титулами!

## ✨ Возможности

### 🎮 Режимы сложности
- **🟢 Лёгкий** - Опыт x1, враги слабые
- **🔵 Нормальный** - Опыт x3, стандартные враги
- **🔴 Сложный** - Опыт x5, сильные враги
- **⚫ Реалистичный** - Опыт x10, **ТОЛЬКО 1 ЖИЗНЬ!**

### 📈 Система прогресса
- Получайте опыт за сообщения в чате
- Повышайте уровень (формула: уровень × 100 опыта)
- Улучшайте характеристики при повышении уровня
- Получайте классовые очки каждые 10 уровней

### 🎁 Ежедневные бонусы
| День | Награда |
|------|---------|
| 1 | 100 монет |
| 2 | 200 монет |
| 3 | Зелье исцеления |
| 4 | 400 монет |
| 5 | Редкая карта |
| 6 | 600 монет |
| 7 | Классовое очко + Легендарная карта |

- Бонус за 30 дней стрика: титул "Верный игрок"
- Сброс при пропуске дня

### 👥 Реферальная система
- Пригласи друга → получи **500 монет + редкую карту**
- Друг получает **200 монет**
- За 5 друзей → титул **"Рекрутёр"**

### 🏆 Турниры
- Создавайте PvP турниры
- Взнос: 1000 монет
- Призовой фонд от взносов участников
- Уникальные титулы победителям

### 📜 Квесты
- **Ежедневные**: "Убей 5 мобов", "Пройди подземелье"
- **Еженедельные**: "Победи в PvP 10 раз"
- Награды: опыт, монеты, карты

### 🏆 Титулы
- Разблокируйте за достижения
- Дают бонусы к характеристикам
- Примеры:
  - "Новичок" (уровень 10)
  - "Опытный воин" (100 PvP побед)
  - "Повелитель Башни" (10 башен)
  - "Легенда" (уровень 500)
  - "Монарх" (уровень 1000)

### 🎁 Промокоды
- `STARTER2024` - Стартовый бонус
- `UPDATE50K` - Бонус к обновлению
- `BIRTHDAY` - День рождения бота
- Создавайте свои промокоды (админ)

### 🏰 Гильдии (Кланы)
- Создавайте свою гильдию
- Вступайте в существующие
- Чат гильдии
- Совместные цели и вклады
- Рейтинг гильдий

### 🏪 Рынок и Обмен
- Продавайте карты другим игрокам
- Покупайте редкие карты
- Обменивайтесь картами с друзьями

### 👤 Редактирование профиля
- **Цитата** - добавьте свою фразу (до 150 символов)
- **Аватарка** - установите фото профиля
- **Дата создания** - отображается в профиле

### 📤 Поделиться прогрессом
- Генерация красивого текста для шаринга
- Реферальная ссылка
- Делитесь достижениями с друзьями

## 🚀 Установка

```bash
pip install -r requirements.txt
```

Создайте файл `.env`:
```env
BOT_TOKEN=your_telegram_bot_token_here
ADMIN_IDS=your_telegram_id
```

Запуск:
```bash
python run.py
```
